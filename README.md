## React dependencies installation

### Environment: Ubuntu 22.04

#### nvm

```bash
sudo apt install curl
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash
source ~/.profile
```

#### NodeJS, npm

```bash
nvm install lts/*
node -v
npm -v
```
